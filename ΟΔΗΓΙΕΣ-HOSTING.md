## 📋 ΒΗΜΑΤΑ ΓΙΑ HOSTING

### Επιλογή 1: cPanel / File Manager
1. Συνδεθείτε στο cPanel του hosting σας
2. Πηγαίντε στο File Manager
3. Ανοίξτε τον φάκελο `public_html` ή `www` ή `htdocs`
4. Ανεβάστε το αρχείο `index.html`
5. Τέλος! Η σελίδα σας είναι online

### Επιλογή 2: FTP (FileZilla)
1. Κατεβάστε το FileZilla
2. Συνδεθείτε με τα στοιχεία FTP του hosting σας
3. Πλοηγηθείτε στον φάκελο `public_html`
4. Σύρετε το `index.html` από τον υπολογιστή σας
5. Τέλος!

### Επιλογή 3: Netlify / Vercel (ΔΩΡΕΑΝ)
1. Πηγαίντε στο netlify.com ή vercel.com
2. Κάντε εγγραφή (δωρεάν)
3. Κάντε drag & drop το αρχείο `index.html`
4. Παίρνετε δωρεάν domain όπως: `technomare.netlify.app`
