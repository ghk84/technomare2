# Hellenic Maritime Registry Website

A professional bilingual (Greek/English) website for a ship and harbor pier registration company.

## Features

### 🌐 Bilingual Support
- Complete English and Greek translations
- Easy language toggle button in the header
- All content available in both languages

### 📄 Pages

#### Home Page (/)
- Company introduction and overview
- Key services highlighting
- Why Choose Us section with company benefits
- Links to both registration services

#### Ship Registration (/naval)
- Comprehensive ship registration information
- List of services offered
- Registration requirements
- Photo gallery with 20 maritime vessel images
- Interactive lightbox gallery with navigation

#### Harbor Pier Registration (/piers)
- Harbor pier registration services
- Infrastructure certification details
- Registration requirements
- Photo gallery with 20 harbor and pier images
- Interactive lightbox gallery with navigation

### 🎨 Design Features
- Navy blue color palette throughout
- Gradient backgrounds for visual depth
- Responsive design (mobile, tablet, desktop)
- Smooth transitions and hover effects
- Professional maritime theme

### 🖼️ Gallery Features
- Grid layout with responsive columns
- Click to open full-size lightbox view
- Keyboard navigation (Arrow keys, Escape)
- Previous/Next navigation buttons
- Image counter display
- Smooth transitions and animations

### 🧭 Navigation
- Top navigation bar with links to all pages
- Language toggle button
- Footer with contact information
- Consistent header and footer across all pages

## Technology Stack
- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router DOM
- Lucide React (icons)
